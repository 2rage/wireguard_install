# wireguard_install
CentOS Wireguard install script

## Install
Run the script

```
wget https://raw.githubusercontent.com/l-n-s/wireguard-install/master/wireguard-install.sh -O wg_install.sh
./wireguard_install.sh
```

## Requirements

Supported distributions:

- AlmaLinux >= 8
- Arch Linux
- CentOS Stream >= 8
- Debian >= 10
- Fedora >= 32
- Oracle Linux
- Rocky Linux >= 8
- Ubuntu >= 18.04
